# 核心防翻车规则（精简版）

完整版在 amazon-image-toolkit 的 anti-pitfall-rules.md。本文件收录所有 prompt 必须执行的高频规则。

---

## 必须执行（每个 prompt 都要检查）

**物理合理性**
- 领夹/夹扣类：必须显示金属夹机构，不能"贴"或"吸"在衣服上
- 插头类：必须真插进端口，零缝隙，USB-C 金属头完全隐藏
- 组件位置：先核对实物照片，咪头/按键/接口的真实位置写进 prompt
- 远景人物：表达"40m距离"时，人物只占画面 1/10-1/8 高度

**人体解剖**
- 避开手指特写按键动作（AI 错误率 30-40%），改用按键发光+箭头
- 人脸尽量剪到下巴/胸口，或用背影/3/4角度/远景小人物
- 必须明确性别，避免"中性脸"

**文字与信息**
- 主标题写 benefit，不写 spec（"Two Voices One Phone" 不是 "2 Mics 1 Receiver"）
- 主标题和图内标签不能信息重复
- 认证/参数标签只用真实可验证的数字

**Prompt 结构**
- 每个 prompt 末尾必须加 STRICTLY AVOID 列表（8-15 项）
- 指定拍摄设备（`shot on Sony A7 with [X]mm lens`）增加真实感
- 金属零件用银/铬色，不用纯金/铜（避免生锈感）
- 有隐藏细节时用"主场景+放大圆圈"复合视角

## v2 方案 B 额外检查

- **T09**：选对手法（广角前伸 vs 极限近景）后，写明具体 lens 参数
- **T10**：反差元素出现的位置是否自然合理（猫趴椅子合理，猫靠近刀片不行）
- **T11**：危机场景是否真实可信；钩子文案是否 ≤6 词；产品引线是否对应实物

## 产品形态一致性（跨图锁定）

在 planning 开始时锁定产品形态描述，所有图（A 和 B 方案）使用完全相同的描述：

```
[产品名]: [颜色] [材质] [形态描述] with [关键零件]. 
[尺寸感参考]. [品牌色/LOGO位置].
```

一旦锁定，不因构图变化而改写。
