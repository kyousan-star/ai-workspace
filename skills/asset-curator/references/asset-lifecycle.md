# Asset Lifecycle

| 状态 | 含义 | 所需证据 |
|---|---|---|
| raw | 仅收集，未评估 | 来源路径 |
| candidate | 可供评审 | 来源、哈希、用途、风险 |
| approved | 用户批准可复用 | 用户、时间、决策记录 |
| published | 已实际上线 | 渠道、URL或后台证据、时间 |
| validated | 经效果或人工复核证明有效 | 指标窗口或明确复核记录 |
| rejected | 已否决 | 否决原因 |
| retired | 不再使用 | 替代资产或退役原因 |

`approved`、`published`、`validated`、`retired` 都必须有人类决策证据。
